This application helps you analyze your chat history to better understand your romantic interactions. Many people find it difficult to read between the lines in text messages. This tool sorts through your conversation data and presents insights in a way that makes sense. It focuses on communication styles, engagement levels, and tone to help you see patterns in how you and your partner talk to each other.

## 🛠 What this tool does

The software scans your exported chat history files and runs them through a specialized analysis engine. It calculates response times, sentiment shifts, and frequency of specific topics. 

- Analyze message sentiment over time.
- Identify changes in conversation depth.
- View engagement heatmaps for different days.
- Compare language patterns between users.

You do not need to understand how the code works to use the application. The system handles all the heavy lifting in the background while you review the results.

## ❓ Common questions

**Do I need a specific file format for my chat logs?**
The software accepts standardized text files exported from WeChat. If you need help with the export process, look for the "Export History" option within your messaging app settings.

**The analysis seems slow, is this normal?**
Large chat files take more processing time. Ensure no other heavy programs run in the background while you process your logs to keep the application moving quickly.

**What if the software shows an error?**
Most errors occur if the imported file is missing data or uses an unsupported format. Check that your export file is complete and try importing it again. If the problem persists, restart the application and verify that your system has enough available memory.

**Can I analyze multiple conversations at once?**
Currently, the tool supports single-conversation analysis to ensure accuracy. You can run new files one after the other to compare different interactions.

## 📝 Troubleshooting guide

If you experience issues, try these steps first:

3. Move your chat log file to your Desktop to ensure the application has proper permissions to read it.
4. If the interface does not load properly, uninstall the software and reinstall the latest version from the link provided above. 

This process often resolves minor glitches caused by installation paths or restricted folder access.
